## Book App

Primeiro, para instalar as dependências::

```bash
yarn
```


Para rodar a aplicação local:

```bash
npm run dev
# or
yarn dev
```

https://www.figma.com/file/KFElqzD983WNyvMY1SaF0c/book-app?node-id=0%3A1

Abra [http://localhost:3000](http://localhost:3000) no seu browser
